完全开源

nRF52832-MDK 是一个开源项目，所有硬件设计资料、源代码、开发文档完全开源，项目托管在 GitHub 上，由社区共同维护。

GitHub »

nRF52832 SoC

nRF52832 是 Nordic Semiconductor 新一代高性能的低功耗蓝牙系统级芯片，ARM® Cortex™-M4F 内核，512kB Flash + 64kB RAM，支持 BLE, ANT, 2.4GHz 协议栈。

了解更多 »

集成多功能调试器

nRF52832-MDK 集成了多功能调试器 DAPLink，即插即用，可实现 U 盘直接“拖放”下载固件、串口调试、CMSIS-DAP 仿真调试。

前往 DAPLink 项目 »

支持多种开源软件

nRF52832-MDK 可以支持多种开源软件，包括 nRF5 SDK, mbed OS 5, Zephyr, Contiki, JerryScript, Espruino, Arduino, Eddystone, Physical Web 等。

跨平台开发

使用开源工具链, nRF52832-MDK 可实现跨平台开发，满足 Mac OSX, Linux, Windows 开发者的开发需求。

提供在线文档

nRF52832-MDK 提供丰富的在线文档，使用 MkDocs 生成文档，由 GitHub Pages 提供支持，方便开源爱好者一起维护。
